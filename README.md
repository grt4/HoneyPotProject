# HoneyPotProject

# Description

This is an early stage multi-service honeypot designed to be setup on a virtual machine. 

# WARNING
# INSTALL ON VIRTUAL MACHINE
# DO NOT INSTALL ON YOUR LOCAL MACHINE! YOUR DATA MAY BE LOST!

# Installation 

chmod +x requirements
sudo ./requirements

# Setup environment

chmod +x setup
sudo ./setup
